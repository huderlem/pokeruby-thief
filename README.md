# Pokémon Thief Ruby and Sapphire

This is a hack of Pokémon Ruby and Sapphire, which forces the player to steal from trainers. Furthermore, the player's Pokémon do not gain experience or grow levels.

To set up the repository, see [INSTALL.md](INSTALL.md).
